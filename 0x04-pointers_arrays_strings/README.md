# holbertonschool-low_level_programming
## 0x04-pointers_arrays_strings
### 0-reset_to_98.c
This function takes a pointer to an `int` as paramter and updates the value it points to to `98`.
### 1-swap.c
This function swaps the values of two integers.
### 2-strlen.c
This function returns the length of a string.
### 3-puts.c
This function prints a string to `stdout`.
### 4-print_rev.c
This function prints a string in reverse.
### 5-rev_string.c
This function reverses a string.
### 6-puts2.c
This function prints one char out of two of a string.
### 7-puts_half.c
This function prints half of a string.
### 8-print_array.c
This function prints `n` elements of an array of integers.
### 9-strcpy.c
This function copies the string pointed to by `src` including the terminating null byte to the buffer pointed to by `dest`.
