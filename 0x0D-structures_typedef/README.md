# holbertonschool-low_level_programming
## 0x0D-structures_typedef
### dog.h
This header file contains all the prototypes used in this project and defines a new type `struct dog` as well as define a new type `dog_t` as a new name for the type `struct dog`.
### 1-init_dog.c
This function initializes a variable of type `struct dog`.
### 2-print_dog.c
This function prints a `struct dog`.
### 4-new_dog.c
This function creates a new dog.
### 5-free_dog.c
This function frees dogs.
