# holbertonschool-low_level_programming
## 0x03-more_functions_nested_loops
### 0-isupper.c
This function checks for uppercase letter.
### 1-isdigit.c
This function checks for a digit (`0` through `9`).
### 2-mul.c
This function multiplies two integers.
### 3-print_numbers.c
This function prints the digits from `0` to `9`.
### 4-print_most_numbers.c
This function prints the digits from `0` to `9` except for `2` and `4`.
### 5-more_numbers.c
This function prints 10 times the numbers from `0` to `14`.
### 6-print_line.c
This function draws a straight line in the terminal.
### 7-print_diagonal.c
This function draws a diagonal line on the terminal.
### 8-print_square.c
This function prints a square.
### 9-fizz_buzz.c
This function prints the numbers from `1` to `100`, but for multiples of `3` print `Fizz` instead of the number and for multiples of `5` print `Buzz` and for multiples of both `3` and `5` print `FizzBuzz`.
### 10-print_triangle.c
This function prints a triangle.
