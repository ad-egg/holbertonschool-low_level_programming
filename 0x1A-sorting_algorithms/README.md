## 0x1A-sorting_algorithms
### 0-bubble_sort.c
This C function sorts an array of integers in ascending order using the Bubble sort algorithm.
### 0-O
This file contains the big O notations of the time complexity of the Bubble sort algorithm with 1 notation per line: in the best case, average case, and worst case.
