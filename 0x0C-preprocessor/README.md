# holbertonschool-low_level_programming
## 0x0C-preprocessor
### 0-object_like_macro.h
This header file defines a macro named `SIZE` as an abbreviation for the token `1024`.
### 1-pi.h
This header file defines a macro named `PI` as an abbreviation for the token `3.141592653591.
### 2-main.c
This program prints the name of the file it was compiled from.
### 3-function_like_macro.h
This function-like macro `ABS(x)` computes the absolute value of a number `x`.
### 4-sum.h
This header contains a function-like macro `SUM(x, y)` that computes the sum of the numbers `x` and `y`.
