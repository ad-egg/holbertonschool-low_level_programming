# holbertonschool-low_level_programming
## 0x0E-function_pointers
### 0-print_name.c
This function prints a name.
### 1-array_iterator.c
This function executes a function given as a parameter on each element of an array.
### 2-int_index.c
This function searches for an integer.
### 3-main.c
This function is used to compile with the other functions.
### 3-op_functions.c
This file contains 5 functions doing math with integers.
### 3-get_op_func.c
This file contains the function that selects the correct function from 3-op_functions.c to perform the operation asked by the user.
### 3-calc.h
This header file contains all the function prototypes and data structures used by the program.
